# Guía de alta para administración

Entra en **Altas de pantalla** y pulsa **Nueva solicitud**. Completa los cinco pasos y usa **Guardar borrador** para no perder el trabajo.

- Establecimiento: nombre interno, nombre público, contacto, tipo y sector.
- Ubicación: dirección y coordenadas. Puedes copiarlas desde el marcador de tu aplicación de mapas.
- Pantalla: indica equipamiento, Internet y lugar físico de instalación.
- Publicación: decide visibilidad, disponibilidad y restricciones.
- Revisión: comprueba el resumen y corrige los avisos.

Cuando no haya avisos, pulsa **Enviar a revisión**. El responsable revisa y usa **Aprobar y preparar**. Aprobar no crea todavía nada en Xibo: el sistema lo indica expresamente porque la escritura remota está desactivada hasta verificar la API. **Descartar** cierra solicitudes que no continuarán.
