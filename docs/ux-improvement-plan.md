# Plan incremental de UX

1. Mantener el sistema visual y los recorridos públicos existentes; medir accesibilidad, responsive y rendimiento con datos reales.
2. Consolidar el asistente de alta con pruebas de usuarios administrativos y textos de ayuda.
3. Añadir filtros comerciales, ficha imprimible y exportación de selección sin exponer datos técnicos.
4. Introducir permisos por capacidad y doble confirmación para operaciones remotas.
5. Habilitar Xibo únicamente tras verificar Swagger, idempotencia y entorno de prueba.

Las métricas de éxito serán borradores completos al primer intento, tiempo hasta aprobación, errores de validación por campo y ausencia de duplicados.
