# Development Plan

## Phase 0 - Documentation and Structure

- Docker development environment.
- Laravel + React + Inertia base.
- Product, architecture, Xibo, database, Docker, admin and testing docs.

## Phase 1 - Functional Vertical

- Xibo sync into `screens`.
- Public screen network.
- Single advice form at `/asesoramiento`.
- Lead persistence.
- Internal lead email.
- User confirmation email.
- Basic tests.

## Phase 2 - CMS

- Editable landing sections ES/GL.
- FAQs ES/GL.
- Legal texts.
- Contact configuration.

## Phase 3 - Robust Admin

- Dashboard metrics.
- Lead CRM.
- Screen visibility override.
- Manual Xibo sync button.
- CSV export.
- Audit log.

## Phase 4 - Visual Polish

- Refined sea/blue identity.
- Responsive map UX.
- Smooth animations.
- Dark mode polish.

## Out of Scope for MVP

- Payments.
- Advertiser private area.
- Creative uploads.
- Xibo campaign/layout creation.
- Automated scheduling in Xibo.
- Advanced roles.
