# Estado actual de web y administración

La web pública tiene una base moderna, luminosa y coherente con la costa gallega. Sus recorridos principales, mapa, formularios, agradecimiento, FAQ, SEO y bilingüismo ya están implementados. No se inventan métricas y los datos técnicos se quedan en servidor.

El área administrativa cubre CRM, sincronización, CMS y diagnóstico. Esta iteración añade la carencia prioritaria: altas preparadas por personal no técnico, aisladas de la tabla `screens`, con estados y trazabilidad. El dashboard incorpora pantallas totales, disponibles y altas pendientes.

Deuda conocida: permisos granulares, ficha comercial individual y exportación CSV específica de la red; QA visual/manual en navegadores y dispositivos; y validación del Swagger Xibo antes de cualquier escritura.
